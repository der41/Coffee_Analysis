# Daily Coffee Transactions

## Overview
**Daily Coffee Transactions** is a dataset that captures individual coffee shop transactions over a one-year period (March 2024 to March 2025). This dataset provides insights into sales patterns, customer preferences, and payment methods, enabling coffee shop managers to make data-driven decisions efficiently.

## Intended Audience
This project is designed for:
- **Coffee shop managers**, like Sofia, who manage busy neighborhood caf√©s.
- Individuals who track daily sales, manage inventory, staffing, and promotions.
- Users who are business-savvy but not data experts, seeking clear insights without spending hours analyzing data.

## Dataset Description
The dataset includes:
- **Transaction details**: Each record captures the drink purchased, the amount spent, and the payment method used.
- **Time-based data**: Enables analysis of sales patterns by day of the week and time of day.

### Key Questions Addressed
1. How do coffee sales differ across the days of the week and times of the day?
2. What coffee categories drive the most sales, and what payment methods do customers prefer?

## Features
- **Comprehensive Sales Data**: Analyze sales trends over a full year.
- **Customer Insights**: Understand customer preferences for drinks and payment methods.
- **Time-Based Analysis**: Identify peak sales periods to optimize staffing and inventory.

## Use Cases
- **Sales Analysis**: Identify high-performing products and peak sales times.
- **Inventory Management**: Forecast demand for better stock management.
- **Staffing Optimization**: Schedule employees based on sales patterns.
- **Promotional Planning**: Target promotions to boost sales during slow periods.

## Getting Started
### Prerequisites
- Python 3.8 or higher
- Libraries: `pandas`, `matplotlib`, `seaborn` (for data analysis and visualization)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/daily-coffee-transactions.git
   cd daily-coffee-transactions
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

### Example Insights
- **Peak Sales Times**: Mornings (8-10 AM) and afternoons (2-4 PM) drive the most sales.
- **Popular Drinks**: Espresso-based drinks dominate sales.
- **Preferred Payment Methods**: Credit cards are the most common payment method.
